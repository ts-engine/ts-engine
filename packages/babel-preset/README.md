The default babel preset for ts-engine.

## Documentation

Checkout the official docs over at https://ts-engine.dev.

## Getting Started

Install the package

```sh
yarn add --dev @ts-engine/babel-preset
```

Create a babel config file

```js
// .babel.config.js
module.exports = {
  presets: ["@ts-engine/babel-preset"],
};
```
